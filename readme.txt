Christmas countdown

== Description ==

A nice countdown to christmas widget. An example is available here - http://christmascountdownclock.com/

bugs? ideas? email me: wb@christmas0.com

== Installation ==

1. Upload `christmas-countdown` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Christmas Countdown widget (in the 'Widgets' menu) to any sidebar.
4. Set widget width (default is 200 pixels).
